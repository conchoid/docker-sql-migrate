# Docker Images for sql-migrate

This repository contains Dockerfiles to install [sql-migrate](https://github.com/rubenv/sql-migrate) with PostgreSQL 17 SCRAM-SHA-256 authentication support within docker containers. Pre-built images are available at [Docker Hub](https://hub.docker.com/r/conchoid/sql-migrate/).

## Available Tags

- `1.8.0-alpine` - sql-migrate v1.8.0 on Alpine Linux
